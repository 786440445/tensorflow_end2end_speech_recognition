## Usage

At first, please choose the corpus and
```
cd corpus
```

### Training
```
cd trainer
```
and
```
./run_ctc.sh gpu_numebr path_to_config
```

### Restoration & Evaluation
```
cd restore
```
and
```
python restore_ctc.py
```
